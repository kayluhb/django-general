=============================
Django Utils
=============================

An opinionated collection of utilities for Django

Documentation
-------------

The full documentation is at https://django-utils.readthedocs.org.

Quickstart
----------

Install Django Utils::

    pip install django-utils

Then use it in a project::

    import django_utils
